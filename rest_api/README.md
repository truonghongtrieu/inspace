## Description
This is RestAPI project based on Nest.js with TypeORM (sqlite)

## Requirements
NodeJS v14 or later

## Installation
```
npm install
npm run dev
```

### Launch
- Open `localhost:3000` with Postman
- Swagger docs: http://localhost:3000/api

### Documentation
- Markdown: `./docs/api.md`
- Swagger: `./docs/Swagger%20UI.html`
